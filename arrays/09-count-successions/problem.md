# Count the Successions of `v1` and `v2`

## Problem

Write a program to read `N` integers into a static array of maximum size `100`, then read two values `v1` and `v2` and determine how many times the succession `v1`, `v2` appears in the array.

A succession `v1`, `v2` means that `v1` is immediately followed by `v2` in two consecutive positions of the array.

The program must check that `N` is between `1` and `100` before reading the array.

## Example

Input:

```text
7
2 3 2 3 4 2 3
2
3
```

Output:

```text
Number of successions = 3
```
