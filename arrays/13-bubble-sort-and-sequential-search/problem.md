# Bubble Sort Followed by Sequential Search

## Problem

Write a program to read `N` integers into a static array of maximum size `100`, then:

- sort the array in ascending order using the bubble sort method
- search for a value `val` using the sequential search method

The program must display the sorted array and the position of `val` in the sorted array.

If the value does not exist, display `Value not found`.

The program must check that `N` is between `1` and `100` before reading the array.

## Example

Input:

```text
6
5 1 4 2 8 3
4
```

Output:

```text
Sorted array = 1 2 3 4 5 8
Position = 3
```
