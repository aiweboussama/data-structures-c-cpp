# Product of Non-Zero Elements and Count of Strictly Positive Numbers

## Problem

Write a program to read `N` integers into a static array of maximum size `100`, then determine:

- the product of the non-zero elements
- the number of strictly positive elements

If the array contains no non-zero element, display `Product = 0`.

The program must check that `N` is between `1` and `100` before reading the array.

## Example

Input:

```text
6
5 0 -2 3 0 4
```

Output:

```text
Product = -120
Strictly positive count = 3
```
