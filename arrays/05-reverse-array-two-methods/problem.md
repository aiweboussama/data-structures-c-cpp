# Reverse an Array Using Two Methods

## Problem

Write a program to read `N` integers into a static array of maximum size `100`, then reverse the array using two methods:

- Method 1: reverse the array using only one array
- Method 2: reverse the array using two arrays

The program must display the reversed array for both methods.

The program must check that `N` is between `1` and `100` before reading the array.

## Example

Input:

```text
5
1 2 3 4 5
```

Output:

```text
Method 1 = 5 4 3 2 1
Method 2 = 5 4 3 2 1
```
