# Split an Array into Even and Odd Values

## Problem

Write a program to read `N` integers into a static array of maximum size `100`, then split the array into two arrays:

- `T1` contains the even values
- `T2` contains the odd values

The program must display both resulting arrays.

If one of the arrays is empty, display `Empty array`.

The program must check that `N` is between `1` and `100` before reading the array.

## Example

Input:

```text
6
4 7 2 9 8 3
```

Output:

```text
T1 = 4 2 8
T2 = 7 9 3
```
