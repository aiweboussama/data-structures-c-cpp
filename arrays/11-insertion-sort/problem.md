# Insertion Sort

## Problem

Write a program to read `N` integers into a static array of maximum size `100`, then sort the array in ascending order using the insertion sort method.

The program must display the sorted array.

The program must check that `N` is between `1` and `100` before reading the array.

## Example

Input:

```text
5
7 3 9 1 4
```

Output:

```text
Sorted array = 1 3 4 7 9
```
