# Static Array Maximum, Minimum, and Average

## Problem

Write a program that reads a list of integers stored in a static array, then displays:

- the maximum value
- the minimum value
- the average of the elements

The program must:

- use an integer array with a fixed maximum size
- read the number of elements first
- read the elements of the array
- compute the maximum, minimum, and average

## Example

Input:

```text
5
10 3 8 21 4
```

Output:

```text
Maximum = 21
Minimum = 3
Average = 9.20
```
