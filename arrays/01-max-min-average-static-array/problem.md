# Maximum, Minimum, and Average of a Static Array

## Problem

Write a program to read `N` integers into a static array of maximum size `100`, then determine:

- the maximum value
- the minimum value
- the average of the elements

The program must check that `N` is between `1` and `100` before reading the array.

## Example

Input:

```text
5
8 3 12 -1 7
```

Output:

```text
Maximum = 12
Minimum = -1
Average = 5.80
```
