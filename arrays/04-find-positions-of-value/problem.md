# Positions of a Value in an Array

## Problem

Write a program to read `N` integers into a static array of maximum size `100`, then read a value `val` and display all positions where `val` appears in the array.

If the value does not appear in the array, display `Value not found`.

The program must check that `N` is between `1` and `100` before reading the array.

## Example

Input:

```text
6
4 7 4 2 9 4
4
```

Output:

```text
Positions = 0 2 5
```
