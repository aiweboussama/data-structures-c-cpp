# Insert a Node at the Beginning of a Linked List

## Problem

Write a program that creates a singly linked list and uses a procedure to insert a new element at the beginning of the list.

The program should:

1. Create a linked list
2. Display the original list
3. Insert a new node at the beginning
4. Display the updated list

## Example

Output:

```text
Original list: 10 -> 20 -> 30 -> NULL
After insertion: 5 -> 10 -> 20 -> 30 -> NULL
```
