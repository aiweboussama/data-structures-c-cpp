# Create and Display Three Linked List Nodes

## Problem

Write a program to build a simple singly linked list with exactly three nodes.

The program should:

1. Create three nodes
2. Assign an integer value to each node
3. Link the nodes together in order
4. Display the linked list from the first node to the last node

## Example

Output:

```text
Linked list: 10 -> 20 -> 30 -> NULL
```
