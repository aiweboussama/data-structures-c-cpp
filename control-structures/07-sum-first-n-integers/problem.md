# Sum of the First N Integers

## Problem

Write a program to calculate the sum of the first `N` integers:

```text
1 + 2 + 3 + ... + N
```

The value of `N` must be a strictly positive integer. The program must check this condition before calculating the sum.

## Example

Input:

```text
5
```

Output:

```text
Sum = 15
```
