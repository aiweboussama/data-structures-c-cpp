# Swap Three Integers

## Problem

Write a program to swap three integers in two directions using two different methods:

1. Using a temporary variable
2. Without using a temporary variable

Do both directions:

- Left rotation: `a -> b`, `b -> c`, `c -> a`
- Right rotation: `a -> c`, `b -> a`, `c -> b`

## Example

Input:

```text
a = 1, b = 2, c = 3
```

Output:

```text
Left rotation: a = 2, b = 3, c = 1
Right rotation: a = 3, b = 1, c = 2
```
