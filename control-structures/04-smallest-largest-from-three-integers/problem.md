# Smallest and Largest Number from Three Integers

## Problem

Write a program to find the smallest and the largest number that can be formed using three integers.

Use the three values as digits in the places:

- `100`
- `10`
- `1`

## Example

Input:

```text
a = 3, b = 1, c = 7
```

Output:

```text
Smallest number = 137
Largest number = 731
```
