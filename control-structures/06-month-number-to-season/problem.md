# Month Number to Season

## Problem

Write a program that reads the number of a month and displays the corresponding season using `switch`.

Use the following seasons:

- `12`, `1`, `2` -> `Winter`
- `3`, `4`, `5` -> `Spring`
- `6`, `7`, `8` -> `Summer`
- `9`, `10`, `11` -> `Autumn`

If the month number is not between `1` and `12`, display an error message.

## Example

Input:

```text
4
```

Output:

```text
Season = Spring
```
