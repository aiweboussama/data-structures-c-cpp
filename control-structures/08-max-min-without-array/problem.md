# Maximum and Minimum Without Array

## Problem

Write a program to display the maximum and the minimum of a list of `N` numbers entered by the user, without using an array.

Read the first number first and use it to initialize both the maximum and the minimum. Then continue reading the remaining numbers one by one and update the maximum and minimum when needed.

## Example

Input:

```text
5
8 3 12 -1 7
```

Output:

```text
Maximum = 12
Minimum = -1
```
