# Copy Shop Total Price

## Problem

Write a program to calculate the total price of photocopies.

The copy shop charges:

- `2 DA` for each of the first `10` copies
- `1.5 DA` for each of the next `20` copies
- `1 DA` for each additional copy

## Example

Input:

```text
35
```

Output:

```text
Total price = 55.00 DA
```
